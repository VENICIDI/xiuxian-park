# 修仙游乐园（Xian Park）

## 一句话介绍（Pitch）

随着灵气浓度增加，修炼突破再无门槛，整个修仙界突然开始流行追求刺激，各大仙门纷纷建设游乐园吸引修士。作为一名落魄园主，你需要利用有限的灵地，不断建设离谱又危险的修仙娱乐设施，让修士们花灵石、刷爽感，甚至心甘情愿渡劫。

## 游戏循环（30 秒一轮）

```text
选择建筑
    ↓
放置建筑
    ↓
建筑联动计算
    ↓
游客营业
    ↓
获得灵石
    ↓
升级/购买新建筑
    ↓
下一天
```

完全不需要复杂操作。玩家始终思考一个问题：

> 怎么摆收益最高？

## 世界观

修仙界和平太久，修士突破越来越简单。

大家觉得：

> 修炼？太无聊。

现在流行的是：

> 刺激修仙。

于是，御剑过山车、雷劫跳楼机、心魔鬼屋、丹药盲盒全部爆火。

你负责经营一家刚开业的小游乐园，目标是成为天下第一刺激乐园。

## 地图

地图尺寸为 8×6，共 48 格。

```text

□□□□□□入口
□□□□□□□□
□□□□□□□□
```

固定内容：

- 入口
- 出口
- 道路

其余格子均可放置建筑。

## 建筑分类

建议仅保留四类建筑，方便玩家理解。

### 1. 游玩设施（赚钱）

示例：

- 御剑过山车
- 雷劫跳楼机
- 飞升大摆锤
- 黄泉漂流
- 剑阵迷宫
- 心魔鬼屋
- 灵兽碰碰车

主要负责赚取灵石。

### 2. 商店（消费）

示例：

- 仙丹铺
- 灵酒馆
- 法宝纪念品
- 孟婆奶茶
- 灵宠零食店

游客经过时产生额外消费。

### 3. Buff 建筑

示例：

- 悟道台
- 聚灵阵
- 雷池
- 护山大阵

主要用于强化周围或特定类型的建筑。

示例效果：

> 悟道台：周围四格建筑收益 +20%。

### 4. 功能建筑

示例：

- 厕所（五谷轮回阵）
- 休息亭
- 医疗站
- 保险公司

通常不直接赚钱，但可以避免游客提前离园或增加游客数量。

## 游客

游客仅保留少量属性：

- 名字
- 门派
- 刺激值
- 钱包
- 满意度

游客类型与偏好：

- 剑修：喜欢速度、飞行和剑。
- 丹修：喜欢商店、美食和丹药。
- 佛修：喜欢悟道和景观，讨厌过度刺激。
- 魔修：刺激越高越开心。
- 妖修：喜欢灵兽。
- 鬼修：喜欢鬼屋。

玩家无需操作游客，游客会自动行动。

## 游客路线

游客沿固定路线移动：

```text
入口
 ↓
 A
 ↓
 B
 ↓
 C
 ↓
出口
```

游客经过建筑时触发建筑效果，然后继续前进。无需复杂 AI。

## 建筑联动（核心）

- 聚灵阵：周围四格建筑收益 +15%。
- 雷池：所有雷系建筑收益 +30%。
- 悟道台：游客停留次数 +1，即可额外游玩一个建筑。
- 灵酒馆：游客消费后有 30% 概率再次消费。
- 孟婆奶茶：游客忘记疲劳，继续游玩。
- 飞升纪念馆：游客满意度达到 100 时产生额外消费。

玩家真正游玩的内容是建筑组合。

示例组合：

```text
聚灵阵
  ↓
雷池
  ↓
雷劫塔
  ↓
飞升塔
  ↓
保险公司
```

这些建筑可以互相增益。

## 建筑品质

品质分为：

- 白
- 绿
- 蓝
- 紫
- 橙

以雷劫塔为例：

- 白色：收益 10。
- 橙色：收益 40，并且雷系加成翻倍。

## 随机建筑（三选一）

每天结束后，玩家从三个随机建筑中选择一个，例如：

1. 御剑过山车
2. 飞升塔
3. 灵酒馆

采用 Roguelike 机制，让每局体验不同。

## 每日事件

- 天降祥瑞：当日收益 +20%。
- 雷劫提前：所有雷系建筑收益翻倍。
- 宗门团建：剑修游客增加 100%。
- 魔修夜场：鬼屋收益 +80%。
- 仙盟检查：满意度不足时罚款。
- 炼丹炸炉：仙丹铺停业一天。

## 建筑升级

建筑升级只提升数值，不引入复杂操作。

以雷劫塔为例：

- Lv1：收益 10。
- Lv2：收益 15。
- Lv3：收益 25。
- Lv4：收益 40。

## Game Jam 范围控制

严格控制在约 15 个建筑，但保证每个建筑都有独特机制。

| 类别 | 建筑 | 核心效果 |
| --- | --- | --- |
| 游乐 | 御剑过山车 | 基础收益高，剑修偏爱 |
| 游乐 | 九转大摆锤 | 相邻刺激设施越多，收益越高 |
| 游乐 | 雷劫体验塔 | 高风险高收益，受雷系增益 |
| 游乐 | 心魔鬼屋 | 魔修、鬼修额外消费 |
| 游乐 | 黄泉漂流 | 路径越长，收益越高 |
| 商店 | 孟婆奶茶 | 恢复游客疲劳，增加停留 |
| 商店 | 仙丹铺 | 所有游客均可产生基础消费 |
| 商店 | 法宝纪念馆 | 满意度高时产生额外消费 |
| Buff | 聚灵阵 | 周围建筑收益 +15% |
| Buff | 悟道台 | 游客额外游玩 1 次 |
| Buff | 雷池 | 雷系建筑收益提升 |
| 功能 | 五谷轮回阵 | 降低游客提前离园概率 |
| 功能 | 渡劫保险所 | 雷劫失败时仍能获得部分收益 |
| 功能 | 招生广场 | 下一天游客数量增加 |
| 功能 | 护山大阵 | 周围建筑免疫负面事件 |

程序工作量主要集中在数据驱动：每个建筑配置收益、标签和联动效果即可，无需为每个建筑编写大量特殊逻辑。
